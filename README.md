# 电力系统智能优化调度平台
# Intelligent Optimal Dispatching Platform for Power Systems



## 项目简介
本项目面向**电力系统优化、电网经济调度、最优潮流计算**等实际工程与科研场景，基于 Python + MATLAB 混合开发，集成潮流计算、经济调度、优化算法、数据可视化与结果分析功能，形成一套轻量、易用、可扩展的智能电网优化调度工具。
项目定位：
- 电气专业**课程设计 / 毕业设计 / 大创项目**
- 考研复试**项目展示**
- 电网调度、电力系统算法岗**简历项目**



## 项目功能
1. **电力系统基础潮流计算**
   - IEEE 标准节点系统（9/14/30 节点）
   - 牛顿-拉夫逊法、快速分解法潮流
2. **电力系统经济调度（ED）**
   - 考虑机组耗量特性、功率平衡、出力约束
   - 等微增率法 / 线性规划求解
3. **最优潮流计算（OPF）**
   - 以网损最小 / 运行成本最小为目标
   - 支持电压、线路潮流等安全约束
4. **智能优化算法模块**
   - 遗传算法、粒子群算法（PSO）等求解复杂优化问题
5. **数据可视化与结果展示**
   - 机组出力、网损、成本曲线绘制
   - 潮流结果表格化输出
6. **简单易用的调用接口**
   - Python 脚本一键运行
   - MATLAB 算法核心可独立调用与扩展

## 技术栈
- 核心语言：**Python**、**MATLAB**
- 数值计算：NumPy、Pandas
